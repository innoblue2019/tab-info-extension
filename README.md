# Chrome 頁籤資訊顯示器

這是一個簡單的 Chrome 擴充功能，可以顯示所有開啟頁籤的詳細資訊。

## 功能特色

- 📊 **分頁使用統計**：記錄並分析分頁使用情況（NEW!）
  - 自動記錄每日開啟/關閉的分頁數量
  - 折線圖顯示趨勢變化
  - 支援 5 種時間範圍：近7日、1個月、3個月、6個月、1年
  - 顯示總開啟數、總關閉數、平均每日開啟數、目前分頁數
  - 資料自動保留 1 年
- 📊 顯示所有頁籤的總數
- 🔔 **重複網域提醒**：自動偵測重複的網域分頁
  - 當開啟新分頁時，如果該網域已有 2 個以上分頁會跳出通知
  - 通知包含兩個按鈕：「查看現有分頁」和「30分鐘內忽略」
  - 點擊通知可快速切換到現有分頁
  - 支援白名單設定，常用網站不會觸發提醒
  - 可暫時忽略特定網域 30 分鐘
- 📥 **下載功能**：將頁籤清單匯出為 txt 文字檔
  - 包含第 x 筆、標題、網址
  - 支援搜尋和排序後的結果下載
  - 群組模式會保留群組結構
  - 自動產生帶時間戳記的檔名
- 🔍 **搜尋功能**：即時搜尋頁籤標題
- 🔄 **排序功能**：支援四種模式
  - 預設排序：瀏覽器原始順序
  - 標題排序：按標題字母順序
  - URL排序：按網址字母順序
  - **合併群組**：按網域(domain)自動分組
- 📁 **群組檢視**：相同網域的頁籤會群組在一起，可摺疊/展開
  - 群組按頁籤數量由多到少排序
- ⚙️ **設定面板**：管理統計、白名單和其他選項
- 📋 列出每個頁籤的詳細資訊：
  - 標題
  - URL
  - 網站圖示
  - 狀態（使用中、已釘選、播放音效等）
  - 視窗 ID 和頁籤 ID
- 🖱️ 點擊任一頁籤可快速切換到該頁籤
- 🎨 美觀的漸層設計和流暢的動畫效果
- 📏 寬螢幕設計（700px 寬度）

## 安裝步驟

1. 下載此擴充功能的所有檔案
2. 開啟 Chrome 瀏覽器
3. 在網址列輸入 `chrome://extensions/` 並按 Enter
4. 在右上角開啟「開發人員模式」
5. 點擊「載入未封裝項目」
6. 選擇包含此擴充功能檔案的資料夾
7. 安裝完成！擴充功能圖示會出現在工具列

## 使用方法

### 📋 基本操作

1. 點擊工具列上的擴充功能圖示
2. 彈出視窗會顯示所有頁籤的資訊

### 📊 分頁使用統計（NEW!）

1. 點擊「⚙️ 設定」按鈕
2. 在設定面板中可以看到「📊 分頁使用統計」區域
3. **時間範圍**：點擊不同按鈕切換統計期間
   - 近7日、近1個月、近3個月、近6個月、近1年
4. **折線圖**：
   - 綠線表示每日開啟的分頁數
   - 紅線表示每日關閉的分頁數
   - 可以看到使用趨勢變化
5. **統計數字**：
   - **總開啟**：所選期間內開啟的總分頁數
   - **總關閉**：所選期間內關閉的總分頁數
   - **平均每日開啟**：平均每天開啟幾個分頁
   - **目前開啟**：現在有多少個分頁開啟中
6. **資料收集**：
   - 從安裝擴充功能後開始自動記錄
   - 每次開啟/關閉分頁都會記錄
   - 資料保留 1 年，超過自動刪除

### 🔔 重複網域提醒

**自動提醒**：
- 當你開啟新分頁時，如果該網域已有 2 個以上分頁，會跳出通知
- 通知會顯示：「已有 X 個 domain.com 分頁開啟」

**通知操作**：
- **按鈕 1「查看現有分頁」**：立即切換到該網域的第一個現有分頁
- **按鈕 2「30分鐘內忽略」**：接下來 30 分鐘內開啟此網域不會再提醒
- **點擊通知本身**：也會切換到現有分頁

**白名單設定**：
1. 點擊「⚙️ 設定」按鈕
2. 在白名單區域輸入網域（例如：google.com）
3. 點擊「➕ 新增」
4. 白名單中的網域不會觸發重複提醒
5. 可隨時點擊「×」移除白名單項目

### 📥 下載功能

3. **下載功能**：點擊右上角的「📥 下載」按鈕
   - 會將當前顯示的頁籤清單匯出為 txt 文字檔
   - 檔名格式：`tabs_YYYY-MM-DD_HH-MM-SS.txt`
   - 包含完整資訊：第 x 筆、標題、網址
   - 如果是群組模式，會保留群組結構

### 🔍 搜尋與排序

4. **搜尋功能**：在搜尋框中輸入關鍵字，即時過濾標題
   - 點擊 ✕ 按鈕可清除搜尋
   - 下載時會包含搜尋關鍵字資訊
5. **排序功能**：使用排序按鈕切換排序方式
   - **預設排序**：按照頁籤在瀏覽器中的原始順序
   - **標題排序**：按照頁籤標題的字母順序排列
   - **URL排序**：按照網址的字母順序排列
   - **合併群組**：按照網域(domain/host)自動分組顯示
     - 相同網域的頁籤會群組在一起
     - 群組按頁籤數量由多到少排序（最多頁籤的群組在最上面）
     - 顯示網域名稱和該網域的頁籤數量
     - 點擊群組標題可摺疊/展開
     - 下載時會以群組格式輸出
6. 點擊任一頁籤項目可以切換到該頁籤

## 檔案說明

- `manifest.json` - 擴充功能設定檔
- `popup.html` - 彈出視窗的 HTML 結構
- `popup.js` - 主要程式邏輯
- `styles.css` - 樣式設計
- `icon16.png`, `icon48.png`, `icon128.png` - 擴充功能圖示

## 權限說明

此擴充功能需要以下權限：
- `tabs` - 用於讀取頁籤資訊
- `notifications` - 用於顯示重複網域提醒通知
- `storage` - 用於儲存白名單和暫時忽略設定

## 技術細節

- 使用 Manifest V3（最新版本）
- 純 JavaScript，無需外部函式庫
- 響應式設計
- 支援中文介面

## 自訂調整

你可以根據需求修改：
- `styles.css` - 調整外觀和顏色
- `popup.js` - 增加或修改顯示的資訊
- `popup.html` - 調整版面配置

## 注意事項

- 此擴充功能只能讀取頁籤資訊，無法修改或關閉頁籤
- 某些特殊頁面（如 chrome:// 開頭的頁面）可能無法完整顯示資訊
